# baidu-pan-delete-repeated-files

删除百度网盘中重复的文件,数据从 mongodb 中获取.这个代码库是`百度网盘批量清理重复文件计划`的一部分。

使用方法

安装 node_modules

```shell
yarn install
```

编译脚本

```shell
yarn build
```

运行脚本

```shell
yarn start
```
